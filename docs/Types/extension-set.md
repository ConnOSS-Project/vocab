# Extension Properties

The extension set comprises the ConnOSS properties that go beyond the core schema.org and CodeMeta properties.

---

## intendedUse

A concise summary of the primary objective or intended use case for the software. Distinct from general description; this focuses on 'why' the software exists and what problems it solves.

| Attribute | Value |
|---|---|
| Range | Text |
| Source | fair4ml |

---

## researchDomain

The discipline, area, or research domain to which this software aligns or belongs to.

| Attribute | Value |
|---|---|
| Range | DefinedTerm |
| Source | ConnOSS |

---

## LegalConsiderations

## ethicalSocialConsiderations
