# Distribution and Packaging

How the software is packaged and delivered. These properties describe download and install endpoints, and package size.

---

## downloadUrl

If the file can be downloaded, URL to download the binary.

| Attribute | Value |
|---|---|
| Range | URL |
| Source | schema.org |

---

## installUrl

URL at which the app may be installed, if different from the URL of the item.

| Attribute | Value |
|---|---|
| Range | URL |
| Source | schema.org |

---

## archivedAt

A link to the archived version or archival location where the original content has been preserved (for CreativeWork, MediaReview, etc.)

| Attribute | Value |
|---|---|
| Range | URL / WebPage |
| Source | schema.org |

---

## encodingFormat

Media type, typically MIME format (see IANA site) of the content e.g. application/zip of a SoftwareApplication binary.

| Attribute | Value |
|---|---|
| Range | Text / URL |
| Source | schema.org |

---

## fileSize

Size of the application / package (e.g. 18MB). In the absence of a unit (MB, KB etc.), KB will be assumed.

| Attribute | Value |
|---|---|
| Range | Text |
| Source | schema.org |
