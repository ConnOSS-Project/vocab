# Software Classification and Type

The classification of the software. These properties describe what kind of software it is and how it relates to other components.

---

## applicationCategory

Type of software application, e.g. 'Game, Multimedia'.

| Attribute | Value |
|---|---|
| Range | Text / URL |
| Source | schema.org |

---

## applicationSubCategory

Subcategory of the application, e.g. 'Arcade Game'.

| Attribute | Value |
|---|---|
| Range | Text / URL |
| Source | schema.org |

---

## applicationSuite

The name of the application suite to which the application belongs (e.g. Excel belongs to Office).

| Attribute | Value |
|---|---|
| Range | Text |
| Source | schema.org |

---

## targetProduct

Target Operating System / Product to which the code applies. If applies to several versions, just the product name can be used.

| Attribute | Value |
|---|---|
| Range | SoftwareApplication |
| Source | schema.org |

---

## codeSampleType

What type of code sample: full (compile ready) solution, code snippet, inline code, scripts, template. Supersedes sampleType.

| Attribute | Value |
|---|---|
| Range | Text |
| Source | schema.org |

---

## featureList
